## 1. 项目是什么

### 1.1 一句话

**刻画 2015-2025 年中国互联网集体叙事系统的相变结构。**

不是预测下一个爆款梗, 不是分析某个梗为什么火。
是回答: 整个叙事生态系统在十年的时间尺度上,
经历了什么样的**结构转变**, 以及**为什么会转变**。

### 1.2 基本世界观

把互联网上的梗和叙事看作一个**物理系统**:

- 每个梗是一个粒子
- 梗之间有相互作用 (变异、竞争、替代、融合)
- 外部世界 (经济、政策、技术、平台) 是施加在这个系统上的**场**
- 系统的集体行为涌现出"相区"——不同的宏观状态

### 1.3 核心发现 (执行摘要)

| 发现 | 证据 | 含义 |
|------|------|------|
| 叙事状态是低维的 | 18维→10维, 90%方差 | 系统有简约描述 |
| 月度轨迹不可预测 | VARX R²=-0.32, lag-1 R²=+0.44 | 点预测是错的, 系统近似随机游走 |
| 存在 3 个物理相区 | GMM+RQA | Origin / Fixation / Peak |
| Fixation 是真实的分离结构 | RQA 零跨相区复发 | 不是聚类 artifact |
| 不可逆性来自外部场漂移 | Time-Reversal 对称 | 不是系统自己坏了, 是外部世界变了 |
| AI/Tech 轴十年来定义可达路径 | Control Manifold | 技术话语重塑了叙事地形 |
| 当前处于亚稳态 | Inertia=0.77, Resilience=0.37, Sensitivity=0.56 | 锁死 37 个月, 越来越容易炸 |

## 2. 数据有什么

### 2.1 三块数据

**外部场 (External Field)** — 51 个关键词 × 132 个月的 Google Trends 数据 (2015-2025)
- 代表"社会上正在发生什么": 经济 (就业、房价)、技术 (AI、ChatGPT)、
  政策 (体制内、专家建议)、文化 (B站、二次元) 等
- 不是叙事本身, 是**叙事发生的外部环境**

**叙事 (Narratives)** — 57 条梗的完整传播学档案
- 22 条来自 B站视频的 LLM 抽取
- 36 条来自 Google Trends 曲线的自动生成
- 每条包含: 传播阶段、变异记录、语义漂移、社会背景

**实时采集 (Scraper v2.0)** — 微博50 + 百度50 + 知乎30 = 130 条热搜/小时
- mote-home 服务器 24/7 运行
- v2.0: 全量 384 维 headline embedding (paraphrase-multilingual-MiniLM-L12-v2)
- 日级聚合: 梗语义相似度 + 注意力集中度 + 语义新颖度
- ⚠ 数据审计发现 (2026-07-05): scraper 数据历史上从未进入任何分析模块。
  所有 127 个月的科学结论仅基于 Google Trends + Narrative JSON。
  v2.0 已修复采集端, 但下游月度分析链尚未接入 (2026 年数据悬空)。

### 2.2 为什么这个数据组合是完整的

```
外部场 u(t) ──→ 提供"外面在发生什么"的慢变量
叙事档案     ──→ 提供"每个梗的一生"的结构信息
实时采集     ──→ 提供"此刻什么在动"的快变量
```

三个时间尺度: 年 (外部场) + 月 (叙事) + 时 (采集)

### 2.3 信号质量危机 (2026-07-06 新增)

一个根本性的本体论问题:

系统采集的所有数据（Google Trends、Scraper headline、叙事 JSON、LLM 概念分数）
测量的不是"现实结构"，而是**人群对现实结构的叙事化反应**。

```
互联网叙事活动 = 对现实结构的叙事化反应
                 + 社会环境预设
                 + 情绪化发泄
                 + 道德直觉判断
                 + 犬儒化消解
                 + ...
                 ≠ 对现实结构的观测
```

这不是测量误差。这是**系统性的**——人群在面对现实结构时,
几乎总是产生叙事化反应而非结构分析。该偏差是集体信息处理的内在倾向。

**关键翻转**: 这个"叙事化倾向"不是需要清洗的噪声。它是**要测量的对象**。
那个让一百万人面对同一堵墙却不分析墙、而是发泄、造梗、站队的机制,
就是"集体层面小真实内在的混沌属性"。

**认知螺旋**: 区分"接近结构分析的讨论"和"纯叙事消费"的标准无法预设,
必须从数据中迭代发现。结论会反向注入数据提取层。
这不是"先清洗再建模"的线性流程。

**对 Observation 层的本体论影响**: Observation 层现在需要标注:
我们所观测的不是现实结构, 而是叙事化反应。这两者之间的关系是我们试图刻画的对象,
不能作为前提假设。

## 3. Level 1: 从叙事 JSON 提取硬事实

**回答的问题**: 每个梗的"硬数据"是什么? (不涉及理论解释)

**输入**: 57 条叙事 JSON (`data/processed/narratives/` + `narratives_from_trends/`)

每条 JSON 的结构示例 (躺平):

```json
{
  "meme_name": "躺平",
  "spread_phases": [
    {"phase": "萌芽期", "time_range": "2021年4月-5月", ...},
    {"phase": "爆发期", "time_range": "2021年5月-6月", ...},
    {"phase": "泛化期", "time_range": "2021年7月至今", ...}
  ],
  "mutations": [
    {"name": "躺平学", "time": "2021年5月", "relationship": "理论化延伸"},
    {"name": "反内卷躺平", "time": "2021年6月", ...}
  ],
  "semantic_drift": {
    "original_meaning": "主动降低欲望、减少工作投入...",
    "current_meaning": "泛指一切消极、低努力的生活态度...",
    "drift_direction": "从具体行为策略向泛化态度标签演变"
  },
  "social_context": {
    "triggers": "高房价、低工资、996...",
    "backlash_events": "《光明日报》等官媒发文批评..."
  }
}
```

### 3.1 四类硬事实

**Stage Occupancy (阶段占比)** — 5 维, 已有
- 用 Google Trends 数据锚定每个梗的五阶段时间边界
  (origin/emergence/peak/controversy/fixation)
- 每月聚合: 当前活跃梗在五个阶段的占比
- 来源: `stage_occupancy.py` → `stage_occupancy.json`

**Mutation Occurred (是否变异)** — 布尔值 → 月度比率
- 检测叙事的 `mutations` 数组是否非空
- 解析变异发生时间, 向前延展 6 个月的"活跃窗口"
- 每月聚合: 活跃梗中有变异信号的比率
- 来源: `narrative_hard_facts.py`

**Institutionalized (是否被制度化收编)** — 布尔值 → 月度比率
- 关键词匹配: `social_context` 中是否出现
  人民日报/新华社/央视/官方/国务院等 20 个机构关键词
- 一旦被收编, 该梗在后续所有月份标记为 True
- 来源: `narrative_hard_facts.py`

**Semantic Drift Distance (语义漂移幅度)** — 标量
- 用 `sentence-transformers` (`paraphrase-multilingual-MiniLM-L12-v2`)
  对 `original_meaning` 和 `current_meaning` 编码
- 计算余弦距离: d = 1 - cos(orig_emb, curr_emb)
- 范围 [0.20, 0.94], mean=0.56 — 有实质性方差
- 来源: `narrative_hard_facts.py`

### 3.2 月度聚合

不是对 57 条叙事直接平均——每个月只有部分梗是"活跃"的。

**活跃定义**: 该梗在当月有 Google Trends 搜索量 > 0。

**权重**: 以 Trends 值为权重, 做注意力加权平均。

**输出**: 127 个月 × (5 维 Stage + 1 维 Mutation + 1 维 Inst + 1 维 Drift) = 8 维
→ `data/processed/level1_hard_facts.json`

### 3.3 为什么叫"硬事实"

这四类特征不涉及任何人类理论——不预设"冲突性应该是一维"、
"虚无退却是负面的"等等。它们就是从叙事 JSON 中直接提取的
**可观测的结构属性**。

## 4. Level 2: 学习叙事状态表示

**回答的问题**: 系统的内在状态是什么? 能用几个维度描述?

### 4.1 特征组装

将 Level 1 的 8 维 + 外部场 PCA 8 维 + 注意力结构 2 维 (HHI + Entropy)
= **18 维特征矩阵** (127 月 × 18)。

**为什么加外部场和注意力?**
Level 1 只描述了叙事本身的结构——但叙事不是在真空中演化的。
外部场 (经济/技术/政策) 和注意力 (热搜集中度/多样性)
是叙事生态的"气候条件"。

### 4.2 PCA 降维

```
18 维 → StandardScaler → PCA → 10 维 Narrative State x(t)
```

**为什么选 10 维**: 保留 90% 方差的最少维度数 (H1a 验证)。
d95=12, d99=14。10 维是在简约性和保真度之间的最优平衡。

**x(t) 是什么**: 127 个月, 每个月一个 10 维向量。
这个向量就是该月的"叙事状态"——
它编码了当前活跃梗处于哪些阶段、变异程度如何、
被制度化收编的比例、语义漂移的幅度、
以及外部气候和注意力结构的配置。

### 4.3 PC1 揭示了什么

PCA 的第一主成分 (解释最大方差的方向):
```
entropy         +0.41  (注意力多样性)
hhi             -0.38  (注意力集中度)
stage_fixation  +0.36  (僵化阶段占比)
ext_pc_1        +0.35  (外部场第一主成分)
stage_origin    -0.34  (萌芽阶段占比)
```

**PC1 的物理含义**: "注意力结构统一化 + 叙事僵化" vs "注意力分散 + 新梗涌现"。
这是叙事系统最主要的变异轴——不是某个梗火不火,
而是**整个生态是统一的还是多元的**。

### 4.4 H1 验证

| 假说 | 含义 | 测试 | 结果 |
|------|------|------|:--:|
| H1a | 存在低维表示 | PCA 信息损失 | ✅ d90=10 |
| H1b | t 期预测 t+1 期 | Ridge VARX vs lag-1 | ❌ VARX test R²=-0.32 |
| H1c | 当前状态主导未来 | 特征 ablation | ⚠️ State-only R²=+0.14 |

**H1b 被否定的意义** (最重要的发现):

"预测下个月的叙事状态"在物理上是不可能的——
不是因为模型不够好, 而是因为系统在月度尺度上**确实近似随机游走**。
这就像你不能预测分子下一秒的位置, 但可以描述气体的温度和压强。

这个否定结果决定了整个项目的方向: **不预测值, 预测结构**。

## 5. Regime Map: 检测相区结构

**回答的问题**: 叙事系统有哪些不同的"宏观状态"? 当前在哪个?

### 5.1 方法

```
x(t) 10维 → StandardScaler → GMM (Gaussian Mixture Model)
                              → BIC 自动选最优相区数 k
                              → 每月分配相区标签 S(t)
```

GMM 假设每个月的叙事状态是从 k 个高斯分布之一中抽取的。
BIC (Bayesian Information Criterion) 在拟合优度和模型复杂度之间自动平衡。

**BIC 选择 k=4**: 4 个观测簇。

### 5.2 四个观测簇

| 标签 | 月数 | 占比 | 主导阶段 | 时期 | 含义 |
|:--:|:--:|:--:|------|------|------|
| R0 | 4 | 3.1% | peak | 罕见爆发 | 全网注意力瞬间集中在一个事件上 |
| R1 | 39 | 30.7% | origin | 2020-2022 | 高能耗的过热涌现期 (疫情+内卷) |
| R2 | 38 | 29.9% | fixation | 2022.12→今 | 僵化锁死 — 新旧梗都动不了 |
| R3 | 46 | 36.2% | origin | 2015-2019 | 健康涌现期 — 新梗自然出现和消亡 |

### 5.3 转移矩阵

```
      →R0    →R1    →R2    →R3
R0→  0.500   0       0     0.500
R1→   0     0.872   0.051  0.077
R2→   0     0.027   0.973   0
R3→  0.043   0.087    0    0.870
```

**关键发现**:

- R2 (Fixation) 的自持概率 = **97.3%** — 一旦进入, 几乎不可能自行逃脱
- R2 → R1 仅 2.7%, R2 → R3 = 0% — 回不去了
- R3 → R1 有 8.7% — 疫情把系统从健康涌现推入了过热涌现
- 转移矩阵几乎是**单向的**: R3→R1→R2, 有明确的时间箭头

### 5.4 驻留时间

| 相区 | 平均驻留 | 中位 | 最长 | 当前 |
|:--:|:--:|:--:|:--:|:--:|
| R2 | 19 月 | 19 月 | 37 月 | **37 月** (仍在继续) |
| R1 | 7.8 月 | 4 月 | 24 月 | — |
| R3 | 7.7 月 | 3 月 | 25 月 | — |

当前 Fixation lock 已经是**中位驻留时间的 2 倍**。
这是 127 个月数据中的极端值。

### 5.5 从 4 簇到 3 相区 (物理合并)

后续的 Recurrence Analysis 证明:
- R1 和 R3 之间有 10 对跨期复发 (ε-邻域内的状态对)
- R1↔R3 的最小相区间距离仅 1.18
- R2 与 R1/R3 的最小距离分别为 2.24 和 3.38 — 明显更大

**物理解释**: R1 和 R3 本质上**同一个 Origin manifold**,
只是在不同外部条件下的"形变"。R2 (Fixation) 是**真正的结构分离**。

合并后的 3 个物理相区:

| 物理相区 | 包含 | 含义 |
|------|:--:|------|
| **Origin** | R1 + R3 | 叙事健康涌现和自然耗散 |
| **Fixation** | R2 | 叙事锁死, 新梗无法破圈 |
| **Peak** | R0 | 罕见全网爆发事件 |

## 6. Irreversibility Tests: 验证结构真实性

**回答的问题**: R2 是真实的结构分离, 还是聚类的投影 artifact?

这是对 GPT "GMM 可能在连续流形上切 bin" 质疑的直接回应。

### 6.1 RQA (Recurrence Quantification Analysis) — "回去过吗?"

**方法**:
1. 设定 ε (状态空间中的"足够近"阈值, 取最近邻距离的第 10 百分位)
2. 对每个月份 i, 找到所有在 ε 距离内的其他月份 j (至少间隔 12 个月)
3. 统计跨相区的复发对数

**如果 R2 是聚类 artifact**:
R2 中的月份应该与 R1/R3 中的某些月份足够接近 (因为它们本质上在同一条流形上)。

**如果 R2 是真实分离**:
R2 与 R1/R3 之间应该**零复发**。

**结果**:
```
R2 → R1: 0 对
R2 → R3: 0 对
R1 → R2: 0 对
R3 → R2: 0 对
R1 ↔ R3: 10 对 (归一化=0.42)
```

**结论**: R2 是**真实的结构分离**。没有任何 R2 月份与 R1/R3 月份在 ε 距离内。
而 R1 和 R3 之间有 10 对复发 — 它们确实是连通的。

### 6.2 Time-Reversal Test — "倒放电影一样吗?"

**为什么需要这个测试**:
如果动力学有内禀时间箭头 (P(x→x') ≠ P(x'→x)),
那么"不可逆锁死在 R2"可能是系统自身的属性。
如果时间反演对称, 说明锁死来自外部。

**三种测试**:

1. **漂移测试**: 10 维中任何一维是否有显著的非零漂移? → **0/10 维显著, 漂移量 0.08**

2. **置换测试**: 随机打乱时间顺序后, 转移步长是否变化?
   → 原始步长 2.46 ≪ 随机重排 5.37 — 时序有平滑性,
   但这是相邻月自然相近, 不是方向性

3. **余弦对齐**: 连续两步的方向是否一致?
   → 均值 -0.29 (弱均值回归, 不显著) — 没有"惯性沿着某个方向一直走"

**结论**: 逐月动力学**时间反演对称**。

### 6.3 综合: WEAK_IRREVERSIBILITY

```
RQA:        R2 是真实的结构分离 ✅
Time-Reversal: 动力学对称, 无内禀时间箭头 ✅
```

**物理图景**: 系统不是自己坏了——是外部世界 (Control 层) 十年来沿着 AI/Tech 方向单调漂移,
把系统带进了一个新的盆地, 然后就困在里面了。如果外部条件逆转,
系统在物理上**可以**回归——只是过去十年没发生。

## 7. Control Manifold: 找到驱动力

**回答的问题**: 外部世界的什么东西, 塑造了叙事系统能去的地方?

### 7.1 从 51 维到 3 维

```
外部场 51维 → PCA → 8维 → Diffusion Map (非线性流形学习) → 3维 z(t)
```

**为什么要 Diffusion Map 而不是只用 PCA**:
PCA 只能捕捉线性关系。Diffusion Map 保留了数据中的非线性流形结构——
两个月份如果在高维空间中通过一系列中间状态相连,
它们在低维投影中也是近的 (扩散距离)。

### 7.2 三个控制轴

**z₁ (AI/Tech 轴)** — 十年单向漂移, 范围 [-0.62, 0.003]

主导关键词:
- AI (r=-0.68) — 十年来搜索量单调上升
- ChatGPT (r=+0.54)
- B站 (r=-0.30)

**物理含义**: 技术话语的密度。AI 相关搜索量从 2015 年的接近零
增长到 2025 年填满整个注意力空间。这不是一个"事件"——
是**基础设施级别的媒介环境变化**。

**z₂ (多元文化轴)** — 范围 [-0.34, 0.88]
- AI (r=+0.25)
- B站 (r=+0.18)
- ChatGPT (r=-0.17)

**z₃** — 范围被前两个轴主导

### 7.3 核心发现: R2 不在 z(t) 极端区间

**初始假说**: "外部场极端配置 → 叙事系统被逼入 Fixation"

**数据检验**: R2 的质心距全局中心 0.11, **比**其他相区平均 (0.15) 更靠近中心

**但是**: z₂ 轴上 R2 处于 99.2% 分位数 — 在单一控制轴上确实极端

### 7.4 修正后的物理图景

```
不是:  u(t) 极端 → R2 逼迫
而是:  u(t) 沿 AI/Tech 方向单调漂移
       → 系统经过一个 R2-favoring 的盆地
       → 滑进去后就困住了
```

**R2 进入点**: 2020-11, 在 z-space 的原点 (z₁=0.001, z₂=0.000)
— 不是在极端条件下进入的, 而是在一个**中性过渡点**滑进去的。

**类比**: 不是被推到悬崖边缘掉下去的——是沿着一条单向扶梯走,
走着走着就到了一个出不来的楼层。

### 7.5 关键待验证问题

**AI/Tech 是因果 driver 还是 time proxy?**

z₁ 与时间高度共线 (AI 搜索热度十年单调上升)。
需要 counterfactual test: 固定时间=2024,
人为假设 AI 趋势下降, 观察 z(t) 是否移动。

如果不移动 → AI 只是时间标记, 不是驱动力
如果移动 → AI 是真正的 control driver

## 8. FR31 Metrics: 四个控制论指标

**回答的问题**: 当前系统有多稳定? 多容易炸? 在叙事图上的什么位置?

四个指标从 10 维 State x(t) 和 5 维 Stage σ(t) 中计算,
每个都有明确的物理含义。

### 8.1 Inertia (惯性) — 0.77

**物理含义**: 势阱深度。系统有多"卡"?

**计算**:
```
Inertia = 0.4 × (fixation% / (origin% + emergence%))
        + 0.3 × (滚动 12 月 lag-1 自相关系数)
        + 0.3 × (fixation 占比本身)
```

- stage 分量 (0.897): origin+emergence = 0%, fixation = 52% → 新梗完全无法破土
- dynamic 分量 (0.858): 状态高度自相关, 变化极慢
- traffic 分量 (0.524): 一半以上的注意力流量在僵化梗上

**>0.65 = "踩刹车也停不下来"**

### 8.2 Resilience (恢复力) — 0.37

**物理含义**: 盆地回归速度。系统被推一下后, 多快能回到平衡?

**计算**:
```
Resilience = 0.3 × 恢复速度 (历史冲击平均恢复时间的反比)
           + 0.3 × (1 - 当前偏离均衡的距离 / 3σ)
           + 0.2 × (1 - 近期冲击频率)
           + 0.2 × 特征值稳定性 (AR(1) 矩阵主特征值)
```

- 当前偏离均衡 3.9σ — 极远
- 2025 年下半年 Resilience 归零 → 系统处于**非平衡态**
- 平均恢复时间仅 3 个月 — 历史冲击恢复快, 但当前偏离太大

**<0.3 = 脆, 一次大冲击可能永久改变状态**

### 8.3 Sensitivity (敏感性) — 0.56 ↑

**物理含义**: Jacobian 谱半径。系统对微小扰动的放大系数。有多容易炸?

**计算**:
```
Sensitivity = 0.35 × Critical Slowing Index (方差比 × 自相关)
            + 0.35 × Perturbation Gain (∥Δx∥ / ∥Δu∥)
            + 0.30 × 方向性敏感度 (>2× 基线方差的维度占比)
```

- Critical Slowing = 0.76 — **分岔前经典预警信号**
  当系统接近临界点时, 方差增大 + 自相关增强同时发生
- Perturbation Gain = 0.26 — 外部场变化目前不直接引起状态变化
  (系统太僵了, 外部推不动)
- 敏感维度: PC4, PC5 > 2× 基线方差

**>0.5 = "一点就炸"的临界态**

### 8.4 Position (图位置)

**物理含义**: 流形坐标。当前在叙事演化图的什么位置?

- 当前: fixation 主导 + peak 次导
- 叙事温度 0.174 (低 — 变异很少)
- 吸引子盆地 #1 (14.2% 历史月份在此, 边缘位置)
- 近 12 月 0 次阶段转换 — 完全静滞

### 8.5 四个指标联合诊断

```
高 Inertia (0.77) + 低 Resilience (0.37)
+ 上升 Sensitivity (0.56) + R2 37 月锁死
= METASTABLE (亚稳态)
```

**物理含义**: 系统处于一个深层势阱中, 内部耗散机制已失效。
但势阱边缘正在变薄 (Critical Slowing 上升)。
一个足够强的外部冲击可能触发**灾变性相变** —
不是渐变回到 Origin, 而是突然跌入一个全新的状态。

### 8.6 微观修正: micro_burst_detector

每小时 130 条 scraper 数据聚合为日度微观指标:
- 话题更替率 (相邻两次采集的话题 Jaccard 距离)
- 注意力集中度 (热搜 Top-10 的 HHI)
- 新颖度 (24 小时内全新话题占比)
- 信号密度 (每天 meme 关键词命中次数)

合成 `micro_burst_score`, 作为 Sensitivity 的快变量修正因子。
当前: WATCH 级别, 修正因子 1.06, Sensitivity 从 0.56 → 0.59。

**多尺度逻辑**: 宏观慢变量决定"在哪个相区",
微观快变量检测"闪洪前兆"。两层同时预警时, 相变概率最高。

## 9. Persona Encoder: 个体投影

**回答的问题**: 一个人的言行, 在叙事图上落在什么位置?

### 9.1 设计原则

**不是分类器, 是测量仪器。**

输出:
- ❌ "你是打工人类型" (点定位)
- ✅ "你的文本在当前叙事图上的投影分布: 打工人 0.51, 普信男 0.23, ..." (概率分布)

### 9.2 方法

**双层编码器** (LLM-free):

```
用户文本 → sentence-transformers (MiniLM-L12-v2) → user_emb
57个梗的描述 → 同一模型 → node_embs
cosine_similarity(user_emb, node_emb_i) → 原始相似度
softmax(相似度 / 0.1) → 概率分布 P(meme | text)
```

**为什么不用 LLM**: 外部 AI 一致反对 query rewriting。
一旦用 LLM 把用户口语改写为"叙事描述风格",
就抹掉了个体与大盘话语之间的**摩擦力** — 这个摩擦力本身就是
最有价值的信号 (口语化程度 = 个体与主流叙事的距离)。

### 9.3 双熔断 — "不知道"是合法输出

**FREE_NOISE (噪声熔断)**: max_sim < 0.30
- 用户文本与所有已知梗都不够接近
- → "你的情境在叙事图上找不到对应结构。系统不做强制匹配。按你自己的判断来。"

**AMBIGUOUS (歧义熔断)**: top2 相似度差距 < 0.03
- 用户文本同时高度亲和两个节点, 无法区分
- → "情况本身具有真实的多义性。拒绝强行选边。不假装读心。"

### 9.4 阈值校准

模型 `paraphrase-multilingual-MiniLM-L12-v2` 对中文跨域文本:
- 自检 (同域, 梗描述片段召回自身): max_sim=0.95, top-1=96.5%
- 用户口语 vs 梗描述 (跨域): max_sim 仅 0.42-0.57
- 纯噪声 (天气等无关文本): max_sim ≈ 0.25

**0.51 vs 0.57 的差异有意义吗?**
在绝对数值上很小, 但在统计分布中代表了沿特定流形方向的投影能量差异。
相对排位和分布尖锐度 (熵) 比绝对值重要。

### 9.5 实例验证

| 用户输入 | Top-1 | Sim | 状态 |
|------|------|:--:|------|
| "老板在群里发奋斗鸡汤, 我想阴阳怪气" | 打工人 | 0.51 | WEAK_SIGNAL |
| "对方聊利益分配就抠字眼阴阳怪气" | 普信男 | 0.57 | WEAK_SIGNAL |
| "不想努力了, 也不想卷了" | 躺平/轻松绷住 | 0.42 | MELTDOWN (歧义) |
| "今天天气不错晒了被子" | — | 0.25 | MELTDOWN (噪声) |

## 11. 四层形式化

所有模块收拢到四个本体论层次:

### 11.1 State (状态)

**定义**: 系统的不可直接观测的内部状态。

```
x(t) ∈ M(z(t))  —  叙事状态活在控制流形上
S(t) ∈ {Origin, Fixation, Peak}  —  相区身份
σ(t) ∈ R⁵  —  五阶段占比
```

**关键性质**:
- x(t) 可以降维描述 (H1a ✅)
- x(t+1) 不可点预测 (H1b ❌) — 状态本身近似随机游走
- Fixation 是真实的结构分离 (RQA ✅)
- 不可逆性来自 Control 层的漂移, 非 State 层内禀 (Time-Reversal ✅)

**对应模块**: `representation_state.json`, `regime_map.json`,
`irreversibility_results.json`

### 11.2 Observation (观测)

**定义**: State 在测量空间中的不完全投影。观测从不声称是真相。

```
y(t) = Proj[x(t)] + noise
```

**观测清单**:
- Level 1 硬事实: Stage/Mutation/Inst/Drift
- 注意力结构: HHI (集中度), Entropy (多样性)
- FR31 四指标: Inertia, Resilience, Sensitivity, Position
- 个体投影: P(meme | user_text) — 概率分布, 非点定位
- 微观闪洪: 日度波动率

**红线**: Attention 是响应变量, 不是序参量。
它反映注意力分配, 而非决定相区归属。

**对应模块**: `level1_hard_facts.json`, `metrics.py`,
`persona.py`, `micro_burst_detector.py`

### 11.3 Control (控制)

**定义**: 不进入 State 演化方程右侧, 但定义 State 可达集合的几何边界。

```
x(t) ∈ M(z(t))  正确  —  Control 形变流形
x(t+1) = F(x(t), u(t))  错误  —  Control 不直接推动状态
```

**控制流形**: 外部场 51维 → PCA 8维 → Diffusion Map 3维 → z(t)
- z₁: AI/Tech 话语轴, 十年单向漂移 (因果性待 counterfactual 验证)
- z₂: 多元文化轴
- R2 不在 z(t) 极端区间 — 进入机制是漂移, 非逼迫

**对应模块**: `external_field_2015_2025.json`, `control_manifold.json`

### 11.4 Dynamics (动力学)

**定义**: 状态如何随时间演化, 以及控制如何形变这种演化。

**三层动力学**:

| 尺度 | 对象 | 已知 | 待建 |
|------|------|------|------|
| 宏观 | Regime 转移 P(S(t+1)|S(t), z(t)) | 转移矩阵方向明确, R2 自持 97.3% | MS-AR 条件概率建模 |
| 介观 | State 漂移 x(t+1)-x(t) ~ D(z(t)) | 近似随机游走, 方差由 Control 调制 | GARCH 方差建模 |
| 微观 | 注意力波动 burst(t) | 闪洪检测已跑通 | — |

**对应模块**: `regime_detector.py`, `representation_learning.py`,
`irreversibility_test.py`, `micro_burst_detector.py`

### 11.5 不该跨越的边界

| 跨越 | 为什么不行 |
|------|------|
| Observation → State | Persona 投影不是心理学诊断 |
| State → 精确预测 | H1b 已否定 |
| Control → 因果变量 | 可能是 time proxy |
| Observation → 序参量 | Attention 是响应, 非驱动 |

### 11.6 FR19 → FR31 的适用范围边界

这是整个框架中最重要的否定性约束。

#### 三类覆盖域

所有用户提问可以归入三种类型:

| 类型 | 定义 | FR31 行为 | 示例 |
|:--:|------|------|------|
| **Type A** (完全覆盖) | 集体数据中有充分信号 | 基于 FR19 数据直接回答 | "公开发表反讽言论会怎样?" |
| **Type B** (部分覆盖) | 宏观环境有数据, 个体变量无数据 | 回答宏观约束, 明确标注个体层面超出覆盖域 | "我该不该辞职创业?" |
| **Type C** (不可触达) | 集体数据中完全无信号 | 直接说不知道, 不做推导 | "我跟她表白会怎样?" |

**Type B 是最容易搞错的一类**。

Type B 情境的特点是: 宏观环境有结构 (经济形势、就业市场、行业趋势),
但个体层面的关键变量 (个人能力、家庭储蓄、风险偏好)
不在集体叙事数据中。FR31 的正确行为是:

```
"宏观层面: 当前 [行业X] 的叙事惯性是 Y, 控制流形沿 [方向] 漂移。
 个体层面: 你的个人能力/储蓄/风险偏好不在系统覆盖域内。
 FR31 不能替你回答'该不该'——只能告诉你宏观地形长什么样。
 决定是你的。"
```

**不是直接说"不知道"——Type B 的价值是**把宏观地形画清楚,
让用户自己决定在上面走哪条路。

#### 补充边界: 叙事伪装 (Gemini 提出)

个体可能使用宏观话语符号包裹完全异质的微观动机。
例如: 私聊中说"我也想考公, 安稳一辈子" —
persona 会将其投影到 `[制度化收编/稳定吸引子]`,
但个体的真实动机可能仅仅是"为了逃避当前的导师摧残"。

这属于 **Observation → State 的测量破裂** —
观测到的话语符号不等于个体的真实状态。
应作为不可触达域的第二核心红线。

#### 新增: OOD (模型失效域) — GPT 提出

与覆盖域不同, OOD 不是"数据中没有", 而是**历史规律可能已经失效**。

触发条件:
- 互联网基础设施发生断裂式变化 (新平台替代旧平台)
- 数据源失效 (Google Trends 被替代、推荐机制整体改变)
- 媒介形态质变 (全面 AI Agent 化、搜索行为消失)

当系统进入 OOD 状态时, FR31 的正确行为不是继续基于历史数据输出,
而是**先识别自己进入了模型失效域**, 再明确提示:
"当前媒介环境已发生结构性变化, 历史统计关系的适用前提可能不成立。"

判断标准: 监控 Observation 层的**结构断裂信号** —
若 HHI/entropy 的月度值连续 3 个月超出历史 3σ 范围,
或 scraper 数据源的覆盖率突然下降 >50%, 触发 OOD 预警。

#### 五态输出模型

综合以上, FR31 的每一次个体层输出应标注五种认知状态之一:

| 状态 | 含义 | 示例触发条件 |
|------|------|------|
| **KNOWN** | 在覆盖域内, 置信度充分 | Type A, persona SUPPORTED |
| **PARTIAL** | 宏观可观测, 个体不可触达 | Type B |
| **UNKNOWN** | 完全不可触达 | Type C, persona FREE_NOISE |
| **AMBIGUOUS** | 信号存在但多义 | persona AMBIGUOUS, top2 gap < 0.03 |
| **OOD** | 历史规律可能已失效 | 连续 3 月超出 3σ 或数据源覆盖骤降 |

#### 边界定义的物理补充 (Gemini 提出)

`Institutionalized` 指标具有双重属性:
- **累积值**: 属于 Observation (反映生态已经多僵)
- **发生瞬间**: 属于 Control 层的外生扰动 (官方机构发文是一次 Delta 冲击)

在形式化中明确: Institutionalized 的历史累积 → Observation;
制度收编动作的发生 → Control。

#### FR19 逃逸代理指标 (Gemini 提出)

R2 的锁死是因为 z₁ 轴的单向顶住。逃逸的硬指标不是 ∥Δu∥,
而是 **z(t) 的横向剪切**: 当 z(t) 两步间夹角 > 60°,
且剪切加速度超过历史最大值的 2σ 时, 势阱边缘可能坍塌。
