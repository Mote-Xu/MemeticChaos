# MemeticChaos 框架设计 v3.0

> 两层预测架构。最后更新：2026-06-27

## 科学问题

集体情感系统的演化是否可以预测？

- **不是**预测单个梗，是预测系统级状态
- **不是**二元方向（混沌/秩序），是具体秩序形态
- **两层耦合**：外部场驱动趋势 + 内部叙事调制混沌

## 两层架构

```
Layer 1 — 外部环境场 (可解释的趋势预测)
  输入: 51 关键词平台注意力 (Google Trends 2015-2025)
  角色: 驱动集体情绪的趋势分量
  方法: PCA降维 + RidgeCV 时序预测

Layer 2 — 内部叙事混沌层 (学习到的调制)
  输入: 87 条传播叙事 (22 B站 + 36 曲线生成 + 29 策展)
  角色: 调制外部场，产生不确定性波动
  方法: 类别分布历史 → 混沌残差修正
  状态: 🚧 当前用类别分布代理，Step 2 替换为 LLM 叙事摘要

输出: 秩序形态 = f(混沌轴, 类别分布, HHI, 叙事熵) → 8 聚类命名
```

## 当前实现: order_form_predictor.py (v0.1)

### 特征工程
- 外部场 51维 → PCA 8维 (解释 94.6% 方差)
- 每月特征: PCA(8) + chaos(1) + cat_dist(5) + HHI(1) + entropy(1) + log_attention(1) = 17维
- 时间窗口: lookback=6 月 → 102 维特征
- 目标: [chaos, cat_dist(5), hhi, entropy] = 8维

### 模型性能 (5-fold TimeSeriesSplit, 125样本)

| 指标 | 值 | 解读 |
|------|-----|------|
| Chaos R² (train) | 0.54 | 外部场可解释 54% 训练方差 |
| Chaos R² (backtest) | -0.98 | 外推失败 → 混沌轴是随机游走 |
| Category R² | 0.51 | 类别分布有中等可预测性 |
| **Structure R²** | **0.77** | **注意力结构(HHI+熵)可预测** |
| Internal layer gain | ~0 | 叙事层尚未生效 |
| vs Naive lag-1 | -25% | 月间混沌轴不如直接用上月值 |

### 关键科学发现

1. **混沌轴 = 随机游走**：证实"与混沌共存"——个别月的混沌波动是不可预测的热涨落
2. **秩序形态有结构**：注意力集中度(HHI)和类别熵有可预测的动力学
3. **8 种历史秩序形态**：从"秩序建构·身份认同·集中·单一"到"混沌释放·虚无退却·分散·多元"
4. **内部叙事层 = 0**：软匹配概念瓶颈对中文无效，等待 Step 2 LLM 替换

### 秩序形态输出维度

| 维度 | 说明 |
|------|------|
| 混沌方向 | 秩序建构 / 边界振荡 / 混沌释放 |
| 主导类别 | 解构自嘲 / 攻击发泄 / 虚无退却 / 身份认同 / 纯粹娱乐 |
| 注意力集中度 | 集中(HHI>0.3) / 中等 / 分散(HHI<0.15) |
| 叙事多样性 | 单一(熵<0.8) / 中等 / 多元(熵>1.3) |

6. **64 条分数仅 36 条有 Trends 时间序列**

## 数据流

```
Google Trends (本地, 飞鸟代理) ──→ data/collector/
微博/B站 (服务器, cron)        ──→ data/collector/
叙事生成 (本地, DeepSeek)      ──→ data/processed/
  ↓
order_form_predictor.py ──→ 每周预测报告 (服务器 cron)
  ↓
sync_from_server.sh ──→ 本地查看报告
sync_to_server.sh   ──→ 推送 Trends + 叙事到服务器
```
