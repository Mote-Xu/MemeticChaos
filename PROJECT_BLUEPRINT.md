# MemeticChaos: 项目蓝图

> 最后更新：2026-06-27

## 项目是什么

用 2020-2025 年中国互联网热梗数据，为人类集体情感的混沌属性建模。

Hassabis 猜想驱动：模因是被集体情感/社会压力/语言演化筛选过的系统——有结构，就能被学习。

## 当前阶段

**FR19 — 集体情感状态预测模型。**

从两层数据（外部平台注意力场 + 内部叙事结构）学习集体情感系统的演化规律，预测未来 1-6 个月的主导秩序形态。

## 数据

```
外部场: 51 关键词, 5217 月数据点 (Google Trends 2015-2025)
内部层: 43 梗, 1048 月数据点
叙事:   87 条 (22 B站 + 36 曲线生成 + 29 策展)
混沌轴: 127 个月
```

## 架构

```
外部平台注意力 (51维) ──→ 趋势预测 (可解释)
        +
内部叙事结构 (87条)  ──→ 混沌调制 (从数据中学)
        ↓
集体情感状态 → 未来 1-6 月秩序形态
```

数学框架 (SIR/相图/吸引子) 退到验证层。

## 部署

- **服务器** (mote-home): 微博每小时 + B站/管线每日，24/7 主阵地
- **本地**: Google Trends 通过飞鸟代理，分析 + 精细建模
- 同步: `bash sync_from_server.sh`
