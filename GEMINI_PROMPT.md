# MemeticChaos 项目总结（给外部 AI）

## 项目是什么

MemeticChaos 是一个计算模因学 × 混沌动力学研究项目。目标是：通过分析 2020-2025 年中国互联网热梗的演变，为人类集体情感的「混沌属性」建模，并扩展到个体层面的混沌属性推断。

## 核心理念

不做「下一个热梗是什么」的预测。做的是：
1. 识别集体情感系统的吸引子（哪些情感状态是「稳定」的）
2. 检测相变条件（在什么压力下，集体情绪会从解构自嘲→攻击发泄→虚无退却）
3. 量化热梗作为「局部秩序建立」的生命周期
4. 从外部行为信号反推个体混沌属性剖面（概率性黑箱映射）

## 当前状态（2026-06-26）

### 已完成
- **数据集**：29 个热梗手动策展，5 类别，2020-2025，JSON 格式
- **SIR 模型**：标准 SIR / SIRS（变异复燃）/ 双群体（核心圈→大众扩散）/ 参数拟合 / 混沌熵分析
- **ABM 仿真**：300 Agent 无标度网络，情感传染/回音壁/熵增漂移/混沌投放，5 种角色
- **吸引子检测**：Takens 相空间重构 / 递归图 RQA / Lyapunov 指数 / 盆地检测
- **个体校准器**：遗传算法从外部行为信号反推个体 chaos_position/resilience/role 分布
- **分析层**：生命周期剖面 / 情感弧线分类(5种) / 相变检测(R₀/混沌轴/熵突变)
- **测试**：24/24 通过
- **B站数据**：22 个梗指南视频已下载，Video_to_Text 管道转文字中

### 关键发现
- R₀=1 是模因传播临界相变点
- 2021年混沌轴剧烈负漂(-0.88) → 躺平/普信男时代
- 攻击发泄类最接近绝对混沌(均值-0.62)，身份认同类偏秩序(+0.34)
- 两个清晰吸引子盆地：R₀<1(stable=0.007) 和 R₀>1.9(stable=0.952)

## 哲学基础

对齐「微尘哲学」核心元定律——项目不是对「混沌」的中立建模，而是在承认以下前提下进行：
- 世界默认熵增，混沌无法被消灭
- 生命的道路是与混沌共存、持续建立健康秩序
- 其他主体的「小真实」是不可穿透的黑箱（个体校准器的概率性输出即源于此）

## 参考框架

- 多智能体仿真 — Mesa (备选) + 自定义无标度网络
