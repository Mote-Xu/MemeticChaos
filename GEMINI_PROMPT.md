# MemeticChaos 项目总结（给外部 AI）

## 项目目标

从外部数据（平台注意力场 + 叙事结构）学习中国互联网集体情感系统的演化规律，预测未来 1-6 个月的主导秩序形态。

不是预测下一个爆款梗，是预测集体情绪系统会以什么形态组织自己。

## 当前数据 (2026-06-27)

```
外部环境场: 51 关键词 × 132 月 = 5217 数据点
  类别: 娱乐消费/社会民生/经济商业/科技数码/国际关系/平台流量/社会情绪
  来源: Google Trends 2015-2025 (CN)

内部梗层: 43 梗 × ~100 月 = 1048 数据点
  每个梗有: 月度注意力曲线 + 混沌轴位置 + 5相区类别

叙事层: 87 条叙事
  22 条 LLM 从 B站视频转录抽取
  36 条 LLM 从 Google Trends 曲线形态生成
  29 条人工策展

集体混沌轴: 127 个月 (2015-01 至 2025-07)
  注意力加权平均, 10 次检测到系统级相变
```

## 当前要做的 (FR19)

**目标**：从外部数据学习集体情感系统的演化规律，预测未来 1-6 个月的主导秩序形态。

**不是**：单纯预测混沌轴涨跌（二元方向太粗糙）。预测的是具体的**秩序形态**——集体情绪会以哪一种结构化的行为模式来组织自己。5 相区是中间脚手架，真正的形态从 5D 约束空间中自然涌现。

**两层架构**（来自 grill-me 明确）：

```
外部环境场 (Layer 1):
  十年各大平台热搜数据 (51 关键词 Google Trends 代理)
  不是社会指标，是全民注意力的全貌
  → 驱动趋势，可解释

内部叙事混沌层 (Layer 2):
  小真实本身的混沌属性
  叙事结构 —— 不是残差统计
  87 条叙事 (22 B站视频 + 36 曲线生成 + 29 人工策展)
  → 调制外部场，产生波动和不确定性
  → 不能纯数学建模，只能从数据中学
```

**预测规范**：
- 长期模型基于十年数据建立底层规律
- 近期预测 (1-6月) 用最近 6-12 个月数据
- 输出：带概率分布的状态预测，不是点预测

**问题**：

1. 127 个月够训练吗？什么模型在小样本时间序列上最稳定？

2. 内部叙事层如何量化输入？87 条叙事有结构化 JSON 也有自由文本——统一 embedding？还是提取叙事结构特征（阶段数、变异数、情绪分布）？

3. 验证策略：滚动窗口 time series CV？留最后 12 个月？基线用什么？

4. 两层耦合方式：外部场预测趋势 + 叙事调制混沌波动——这个架构对吗？

## 技术栈

Python + scikit-learn + Google Trends (pytrends via proxy)
服务器: Ubuntu 24.04 (mote-home) 24/7 采集
