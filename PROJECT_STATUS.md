## 0. 一句话

用 2015-2025 年中国互联网热梗数据,刻画集体叙事系统的结构。
**不是预测器**(月度点预测已证伪,H1b)。当前在做:刻画状态空间结构 + 猎杀承重预设。
项目特点:**反复推翻自己的架构,把方法论本身也当研究对象**——每个结论都带 epistemic 等级,承重预设显式化。

## 1. 研究对象(经多次重定义后的当前版本)

测的**不是**"集体情绪状态",也**不是**现实结构本身,而是——
**人群把现实加工成叙事(造梗/发泄/站队)这个转换过程本身的结构。**

那个让一百万人面对同一堵墙、不分析墙、而是造梗的东西 = 集体层面信息处理的内在倾向。这是研究对象。

**第0层地基(人类已定性 2026-07-12 = 定义,不是信念也不是待验假设):**
"叙事化倾向是研究对象;我们 posit 它可被一个『在已知预设边界下覆盖面尽可能广的结构』刻画。"
- 不是**信念**:信念主体导向、私有;而叙事化倾向带**客体导向**成分(什么梗火反映共享世界模型),主体间可及,够格当对象。
- 不是**待验假设**:项目不检验"结构存不存在",而是"在预设边界内找覆盖最广的结构"——存在性被 posit,不被 test。
- ⚠ **守栏(防 Semantic Smuggling)**:定义只到"对象 + 建模姿态",**不把"有稳定结构"焊进定义**(否则稳定性 true-by-construction)。结构的**覆盖度/鲁棒性仍是经验产出**(curation_sensitivity 正在量:d90 鲁棒=有真东西;簇数脆弱=那部分是偶然)。定义给"研究对象合法",数据给"这结构好不好"。
- 落点:整栋楼建在**一条诚实画的线**上——线本身合法(定义可画),线上盖的楼(结构质量)照样要数据承重。

## 4. ★核心发现 + 当前 epistemic 状态★

**这是本文最重要的一节。每个"硬数字"都是真的,但它们的『意义解释』挂在承重预设上——预设一松,解释就悬,但数存活。**

### 4.1 硬的(E1/E2,对 curation 鲁棒或方法自证)
- **低维性 d90≈10**:SUPPORTED,且 **curation 敏感性证明鲁棒**(CV=0.03,换关键词/叙事纹丝不动)。这是最硬的一块。
- **PC2(churn/mutation轴)AR 参数非恒定**:Nyblom p=0.0065(AIC选阶+残差白噪声),TVP-AR 佐证(持久性主根 0.42→0.86,boot_p=0.009)。**低维层不是没功率。**
- **平台生态 16.5× > AI 1.9×**(Counterfactual):平台此消彼长是主驱动,非 AI。
- **H1b REJECTED**:月度点预测不可行(VARX R²=−0.32),状态近似随机游走。

### 4.2 ★被撞松的(承重级,本周新审)★
**`regime_discreteness.py`(null 校准的 gap/Jaccard/Silverman)撞松了"4 相区/R2 盆地"这套:**

- **gap statistic 最优 k=1**(原 regime_detector 的 BIC 只从 k≥3 搜,结构上排除了"没有簇"——隐藏预设)。
- **GMM 4 簇全部自举 Jaccard <0.6**(R2 仅 0.48)——按标准阈值全不稳定,R2 不是稳定锚。
- 唯一非循环的真结构信号 = **PC1 多峰**(Silverman p=0)。
- **裁决:regime-discretization 拆三层**——
  - RD-a 低维有结构(非纯连续)= **holds**
  - RD-b 分成 K 个离散相区 = **suspect**
  - RD-c R2 是离散分离盆地 = **not-supported**
- **诚实边界:127×10 稀疏 → UNDERPOWERED,也没坐实"是连续的"。** 只是"离散相区"没撑住。

**后果(措辞已终审校准 2026-07-12):**
- **"R2 自持 97.3%"作为『R2 相区的属性』——倒。** 测它的代码没固定 k=4,每次让 BIC 选 k(多选 7)、取"最纯 fixation 口袋"当 R2 量自持——是个 **k-无关代理量**,贴了"R2 自持"标签。R2 相区只在 k=4 存在,k=4 是 4% 偶然。
- **那个"自持稳、越细分越接近 1.0"读对了是反盆地、指向连续:** k 越大 fixation 口袋越小越纯、自持越接近 1——"连续高密度区越切越细刨出更黏小口袋"的特征,不是离散吸引子。
- **零复发同理:** 0 复发是事实(存活),但盆地的定义性特征(回拉力)从没被测过;连续慢漂入新区不回头一样给零复发。零复发中性偏**不支持**盆地。
- **唯一活下来的**:E1"状态空间存在一个 fixation 占比高、驻留久的区域",但那是对**连续密度**的描述,不是"有确定自持概率的离散相区"。

### 4.3 简并/未决的(挂在无法辨识的预设上)
- **P2 时不变性 = UNDERPOWERED**(时段审计)。拆:P2a 低维参数平稳=**对PC2 REJECTED** / P2b 全局机制平稳=未决 / P2c 观测算子平稳=未决。
- **PC2 漂移的归因三支简并**:机制真变 / critical slowing / 观测算子变——Nyblom/TVP 都分不开。
- **R2 是什么**:四个 E4 图景并存不选边(结构崩塌/共识收敛/通信协议/叙事阻尼器)+ 新增 drift-dwelling。它们可能是不同统计量的不同投影,未必互斥。

## 5. 方法论基础设施(长期资产,除非内部矛盾)

| 资产 | 作用 |
|---|---|
| **Evidence Ledger**(E0原始/E1统计/E2操作/E3条件解释/E4机制假说 + Assumption列) | 假设倒了只失效挂靠的 E3/E4,E0/E1/E2 数存活。33 条,三桶级联查询。 |
| **四类标注** | 数学工具/统计描述/操作定义/物理声称。物理声称需独立证据,不能同组数据既提又验。 |
| **三层离散化** | 操作离散化(月/PCA/GMM)安全 / 分析离散化(时段切分)安全但属实验设计 / **本体离散化**(五阶段/相区宣称世界本体)危险。 |
| **Competing Explanatory Layer** | 多个 E4 并存,直到统计量杀掉几个。不给任一图景加冕("存在唯一图景"是已证伪的预设)。 |
| **Semantic Smuggling** | 同一组数据既定义又验证(如循环的 R2_axis);或把物理声称冒充观测。持续要抓。 |
| **分辨率墙** | 月度数据对很多问题检验力≈0。承认"仪器答不了"(UNDERPOWERED)是合法结果。 |
| **检测-称重不对称** | AI 建造/局部严谨强,但承重预设的称重不成比例地靠人类。 |

### 5.1 检测-称重不对称 + 别同时发明方法和对象(整合自已删的 PROJECT_LOGIC)

**几乎每一次架构级推翻都是人类发现的,不是 AI。** 更准确:AI 常"看见了但不称重"——检测到基础疑点,随手记成 caveat 脚注、接着往上建;人类把同一疑点称重为"可能致命"、停下来挖。三个机制:① AI 生成偏向(本能是加东西,拆自己脚下的地基逆本能);② AI 因上下文被换掉,每个新会话从文档"再继承"框架、倾向当它给定(人类记得框架是被发明的);③ 称重需要站在楼外,建造者在结构内部最容易把内部当"自然"。

**后果(设计原理,非感慨):人类的理解带宽 = 项目限速器。项目安全速度 = 人类审计速度。** AI 建得比人类审得快,是在没审过的地基上加高——楼越高推翻越贵。**真正的交付物不是代码,是"人类能审计这段逻辑"这件事。**

**配套纪律:别同时发明方法和对象。** 对象(叙事系统的结构)是新的、没成熟先例;那就让每一步的**方法**踩在已验证的成熟石头上(RQA / Nyblom / gap statistic / null simulation…),别自己发明方法。每引入一个东西标清:🪨 借来的成熟石头(安全) vs 🧱 自制的砖(=预设,承重且未验)。

   ↑

**重量落在哪:** 第4层的统计**是硬的**(借来方法算出的 E1/E2),但它们**全部**通过第2-3层的预设才有"意义"。例:R2 自持 97.3% 是真数;"R2 是一个离散相区/盆地"这个解释,吊在第2层预设1 + regime-discretization 上——**而 regime-discretization 本周已被撞松(§4.2)**。**第0-2层最活、最难验、也最承重——上面越漂亮,越依赖这三层没塌。** Evidence Ledger 是这张图的机器可读版(`depends` 列=箭头,`--if-fails <假设>`=抽一根柱子看塌哪些,E0/E1/E2 观测层不陪葬)。
