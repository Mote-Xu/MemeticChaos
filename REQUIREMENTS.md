# 需求文档

## 功能需求

### FR1: 热梗数据集 ✅
- 手动策展 29 个代表性中国互联网热梗（2020-2025）
- 每条记录包含：名称、年份、来源平台、类别、生命周期、情感弧线、混沌属性向量、完整演变叙事
- B站梗指南视频字幕数据作为补充（22个视频，处理中）

### FR2: SIR 模因传播模型 ✅
- 标准 SIR 模型：Susceptible → Infected → Recovered
- 扩展 SIRS 模型：加入变异/复燃机制（σ）
- 双群体模型：核心圈层 → 大众扩散
- 参数提取：β (感染率), γ (恢复率), R₀ (基本再生数), σ (变异率)
- 参数拟合：从时间序列反向拟合 SIR 参数
- 混沌熵分析：香农熵轨迹追踪系统混沌度

### FR3: 生命周期分析 ✅
- 从 SIR 曲线提取 quantitative 生命周期
- 与策展数据 qualitative lifecycle 交叉验证
- 四种模因类型：脉冲型 / 爆发型 / 长尾型 / 流产型
- 跨类别生命周期统计对比

### FR4: ABM 多智能体仿真 ✅
- 300 Agent 无标度网络
- 5 种角色：normal / builder / injector / lurker
- 情感传染 + 回音壁效应
- 熵增漂移 + 混沌投放
- 参数扫描：传染率 / 混沌投放强度

### FR5: 吸引子识别 ✅
- Takens 延迟坐标嵌入（相空间重构）
- 递归图量化分析（RQA）：REC, DET, LAM, ENTR, L_max
- Rosenstein 算法估算最大 Lyapunov 指数
- 吸引子盆地检测
- 混沌/周期/准周期/不动点 四分类

### FR6: 相变检测 ✅
- R₀ 临界跨越检测（传播 on/off 相变）
- 混沌轴年份间漂移检测（集体情感方向变化）
- 熵突变检测（系统混沌度 sudden change）

### FR7: 情感分析 ✅
- 5 种情感弧线分类：rise_fall / fall_rise / inverted_U / U_shape / oscillating
- 情感香农熵计算
- 混沌位置与情感特征的关联分析

### FR8: 个体混沌属性校准 ✅
- 从外部行为观测反推个体混沌参数
- 遗传算法搜索最优参数匹配
- 概率性输出：角色分布 + 参数不确定性 + 置信度
- 4 种场景模板：order_builder / chaos_injector / normal_follower / lurker

### FR9: 可视化 ✅
- SIR 曲线对比 / 混沌景观散点图 / R₀ 扫描图
- 熵轨迹 / 双群体模型 / 情感弧线图
- 可复用 plot 函数库

## 非功能需求

### NFR1: 可复现性 ✅
- 所有分析通过 Python 脚本可复现
- 依赖通过 conda 环境管理
- 测试套件 24/24 通过

### NFR2: 数据质量 ✅
- 策展数据包含完整 narrative 和来源
- 每条热梗包含 chaos_vector + propagation_model

### NFR3: 扩展性 ✅
- 模型支持添加新热梗条目
- SIR/ABM 参数可动态调整
- 个体校准器支持新增行为信号类型

### NFR4: 哲学一致性 ✅
- 混沌 ≠ 随机，追求确定性结构识别
- 热梗 = 局部秩序建立
- 个体校准 = 概率性黑箱映射，不是确定性测量
- 所有输出声明预设和不确定性

## 热梗分类体系

| 类别 | 描述 | 混沌轴位置 |
|------|------|-----------|
| 解构自嘲 | 对自身处境的幽默解构（如"牛马""吗喽"） | 中间偏混沌 (-0.33) |
| 攻击发泄 | 带对立和攻击性的情绪宣泄（如各类对立梗） | 接近绝对混沌 (-0.62) |
| 虚无退却 | 放弃努力的虚无主义表达（如"躺平""四不"） | 接近绝对混沌 (-0.59) |
| 身份认同 | 基于群体身份的意义建构（如"小镇做题家"） | 偏秩序 (+0.34) |
| 纯粹娱乐 | 无明确价值负载的娱乐性传播 | 中性 (+0.19) |
