# 需求文档

## 功能需求

### FR1: 热梗数据集
- 手动策展 20-30 个代表性中国互联网热梗（2020-2025）
- 每条记录包含：名称、年份、来源平台、类别、生命周期、情感弧线、混沌属性向量、完整演变叙事

### FR2: SIR 模因传播模型
- 标准 SIR 模型：Susceptible → Infected → Recovered
- 扩展 SIRS 模型：加入变异/复燃机制
- 双群体模型：核心圈层 → 大众扩散
- 参数提取：β (感染率), γ (恢复率), R₀ (基本再生数), σ (变异率)

### FR3: 生命周期分析
- 热梗从出现到消亡（或固化）的完整生命周期追踪
- 不同类别热梗的生命周期形态分类（爆发型 / 长尾型 / 脉冲型）
- 百度指数/微博热搜时间序列拟合

### FR4: 吸引子识别（后续）
- 热梗语义空间的吸引子检测
- 集体情感的相变点预测
- 在什么条件下集体情绪会从一种模式切换到另一种

### FR5: ABM 多智能体仿真（后续）
- 构建带有情感阈值、连接度、回音壁效应的 Agent 网络
- 验证宏观涌现行为

## 非功能需求

### NFR1: 可复现性
- 所有分析通过 Jupyter Notebook 可复现
- 依赖通过 conda environment.yml 管理

### NFR2: 数据质量
- 策展数据包含完整引用和来源
- 自动化爬虫数据标注来源和时间戳

### NFR3: 扩展性
- 模型设计支持后续添加新的热梗条目
- SIR 模型参数可动态调整

## 热梗分类体系

| 类别 | 描述 | 混沌轴位置 |
|------|------|-----------|
| 解构自嘲 | 对自身处境的幽默解构（如"牛马""吗喽"） | 中间偏混沌 |
| 攻击发泄 | 带对立和攻击性的情绪宣泄（如各类对立梗） | 接近绝对混沌 |
| 虚无退却 | 放弃努力的虚无主义表达（如"躺平""四不"） | 接近绝对混沌 |
| 身份认同 | 基于群体身份的意义建构（如"小镇做题家"） | 中间偏秩序 |
| 纯粹娱乐 | 无明确价值负载的娱乐性传播 | 中性 |
