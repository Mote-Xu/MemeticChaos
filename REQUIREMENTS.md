## 核心发现

1. **混沌轴 = 随机游走**：月间预测 MAE 比 lag-1 差 25%。确认"与混沌共存"不是妥协，是实证。
2. **秩序形态有结构**：注意力结构 R²=0.79, 约束场 R²=0.52。可预测的不是每月混沌值，是系统的形态配置。
3. **机器见 3 类叙事，非人类 5 类**：ARI=0.27。数据自己说了不同的话。
4. **约束场方差 22x 提升**：LLM 概念打分替代软匹配后, Humor std 0.01→0.22, Conflict 0.01→0.20。
5. **内部叙事层贡献仍是 0**：当前最大未解决。约束场方差有了，架构有了，还没转化成预测能力。

## FR31: 情感约束场顾问 — 个体层面约束场建模

> 体系知识底座来源：`E:\Desktop\wctxjx.txt`（用户价值体系完整文档）。
> ⚠️ AlphaGo 原则：体系知识写入 .md 作为背景参考，**不硬编码进模型逻辑**。
> 机器从数据中自己发现结构，不被人类总结的套路污染。

### 要解决的问题

市面上的情感咨询机构（如虹姐）提供的外部视角有价值——时机判断、温度检测、
策略校准。但缺点致命：
1. 不理解用户的认知框架
2. 爹味，自认认知高于用户，无法区分"认知"和"个人选择"
3. 给的建议是通用"泡学"技巧，不与用户自身的认知体系对接

### 核心洞察

集体层面的涌现模式为个体咨询提供**参考基线**——不是推理引擎，是背景知识。
真正驱动 FR31 的是个性化数据。

**三层架构**：
```
Layer 1 (参考层): MemeticChaos 集体规律
  — 约束场动力学的通用原理（信号-动机分离、秩序形态、结构约束）
  — 作为"典型情况下会发生什么"的参考，不直接做个体判断

Layer 2 (个体模型): 对方的画像 + 互动历史
  — 人格特质、偏好、行为模式、沟通风格
  — 线下见面记录、关键事件时间线
  — 聊天历史中的信号密度变化、回复风格偏移

Layer 3 (情境层): 当前上下文
  — 最新聊天记录、你的情感温度、时机约束
  — 她的近期状态（考试/旅游/放假/独处 vs 社交）
```

集体规律提供宏观参考——但要不要在当下推进、怎么推进、用什么语气——
这些只能在数据层和情境层上决定。

### 核心需求

1. **知识底座**：
   - Layer 1 (参考层): MemeticChaos 发现的集体混沌规律
   - Layer 2 (数据层): 她的个体画像 + 你们的互动历史——纯数据
   - Layer 3 (情境层): 当前聊天上下文 + 你的情感温度 + 时机约束

2. **回复生成**：根据当前聊天上下文 + 双方已知特质 + 你的情感温度，生成 1-3 条
   回复建议。每条含：
   - 回复文本
   - 策略理由（为什么这么回，基于对她小真实结构的判断）
   - 风险标注（可能触发她的哪种回避/退缩反应）

3. **时机判断**：不是每次对方发消息都应该立刻回。输出"现在该回/该等/该收尾"。

4. **情感温度检测**：参照你历史上"系统被劫持"的阈值（上次断联前夜的情感强度），
   输出当前你的情感强度是否在安全区间。过热时建议冷却策略。

5. **对方约束场建模**：根据已知信息推断对方当前承受的约束条件（社交压力、家庭
   期待、对情感确认的回避倾向、自我价值感状态等），并据此调整回复策略。

### 体系知识底座

FR31 不是通用情感顾问。以下为背景参考，**不硬编码进代码**：

> ⚠️ 用户完整体系见 `wctxjx.txt`（仅作背景参考，不进代码）。
> 
> 代码中**唯一可引用**的结构划分：
> - 小真实 = 感知(输入) + 认知(处理, 含价值判断/偏好) + 体验(存储)
> - 大真实 = 包含一切的总体宇宙
> - 现实结构 = 主体小真实以外的大真实中的一切
> - 行动 = 小真实与外部世界唯一的桥梁，由小真实驱动
> 
> 用途：建模**对方小真实的内部结构**，尤其是认知层中代表偏好的混沌属性。
> 其余理论（核心元定律、价值排序、三种恶意模型、Fi-Te、利益互换等）**一律不进代码**。

### 关键约束

- **不替你做决定**。给你选项和推理，最终选择永远是你的。
- **不变成爹味教练**。不假装比你更了解你自己或对方。与你的关系是知识/经验提供方，非认知上更高的指导者。
- **忠实于你的体系**。用你的概念工具（利益互换、三种恶意、小真实/大真实、Fi-Te、
  逃避型vs策略型延迟）做推理，不译为通用心理学词汇。
- **如实标注不确定性**。不知道对方的动机就是不知道——量子混沌态恶意不可观测，
  不假装能读心。
- **不过度分析对方**。她不一定是策略型人格，不应过度拟合她的行为。她更多是
  在回应舒适/不舒适，而非执行精密策略。
- **区分认知与个人选择**。你的某些行为（如断联）是情感劫持下的不得已选择，
  而非认知不到位。不将其归为"认知问题"。
- **不追求"善"**。"善"是体系自然运作时的副产品，不是追求的目标。
  追求"善"会滑向绝对秩序——核心元定律明确反对的敌人之一。
